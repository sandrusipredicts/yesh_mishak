## Required manual Play Console sections

- Store presence: main store listing and both locales.
- App content: privacy policy, ads, app access, target audience/content, content rating, Data safety, news-app status if asked, advertising ID if asked, and any health/financial declarations shown by Play.
- Release: internal/closed/production track, countries/regions, release notes, staged rollout, and managed-publishing decision.
- Policy status: review all warnings and blockers immediately before submission because console questionnaires can change.
