# E10-04 completion evidence

This directory separates repository-verifiable evidence from manual Play Console and device evidence.

Do not commit credentials, private tester data, unredacted console exports, or screenshots containing personal information.

Required for completion:

- `asset-validation.md`: commands, date, commit, and validation results.
- `build-verification.md`: frontend, Capacitor, Android resource, and Gradle results.
- `sha256sums.txt`: hashes for approved source, store graphics, Android launcher/splash resources, and final screenshots.
- `capture-manifest.md`: screenshot file/order/build/locale/device provenance.
- `owner-approvals.md`: visual rights/crops, metadata, legal/privacy, Data safety, and release approvals.
- `console-upload-verification.md`: Play Console Preview and upload evidence.
- Build/install evidence linked from the release record.

Console screenshots may be stored in an access-controlled release system and linked here rather than committed.
