# Release assets

This directory contains source-controlled assets and operator-owned metadata for distributable releases. It is intentionally separate from application source code.

- `brand/` contains approved source artwork and provenance.
- `google-play/` contains Google Play listing graphics, localized copy, declaration worksheets, and completion evidence.
- `scripts/` contains deterministic export and validation tools.

Do not add credentials, signing keys, service-account files, Play Console exports containing personal data, or unapproved artwork.
