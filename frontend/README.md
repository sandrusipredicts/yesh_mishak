# yesh_mishak frontend

Minimal React + Vite frontend for local backend connectivity testing.

## Setup

```bash
npm install
```

Create or keep `.env`:

```text
VITE_API_URL=http://localhost:8000
```

Run the dev server:

```bash
npm run dev
```

Open the local Vite URL and confirm the page displays:

```text
Backend status: ok
```
